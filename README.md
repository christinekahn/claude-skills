These are the same skills available as one-click `.skill` downloads on my [Skill Marketplace](https://dreamsanddelivery.com/skills.html).

## Installing a Skill

Each skill is a folder with a `SKILL.md` following Claude's skill format (YAML frontmatter with `name` and `description`, then the instructions).

- **Claude Code**: copy the skill folder into `.claude/skills/` in your project (or `~/.claude/skills/` for all projects).
- **Claude Desktop / Cowork**: add the skill via the skills settings, or download the packaged `.skill` file from the [marketplace](https://dreamsanddelivery.com/skills.html).
